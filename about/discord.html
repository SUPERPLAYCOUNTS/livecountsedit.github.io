The Discord usernames of the owners are ok_aj (Straight From MG), randomperson3465 (RandomPerson3465), and guccipanda69 (Sync). Only DM us if the Discord server/invite above is not available, please ask for help in the Discord server itself.

Rules

Last revised July 8, 2024.
  1. Be respectful to everyone around you. Avoid racism, slurs, discrimination, homophobia, sexism, transphobia, etc.
  2. This is an online community, so avoid sharing any personal data that could be used against you. IP grabbers and other sketchy links are strictly prohibited.
  3. Avoid causing trouble in the server. We don't appreciate trolling, spam, or controversial topics (such as politics).
  4. Advertising is prohibited in this server. Links, self promotion, and shelling out are not allowed.
  5. Do not invite alternate accounts. Any account that is suspected of being an alt or spam account will be kicked. (Owners are exempt from this rule)
  6. Please don't annoy the owners. This means that you should ask for help in this server instead of in direct messages, and do not ping the owners for suggestions or anything off-topic.
  7. Please use common sense. This means that you should use channels correctly, and you shouldn't use loopholes to violate the spirit of these rules.
Member - All members are given this role (except bots and alt accounts of owners)
Regular - Given to active members of the server (has no specific requirements)
Contributor - Given to those that contribute to the website's Github repository
Server Booster - Given to those that boost the server in Discord
Bots - Given to the Discord bots in the server (Dyno, carl-bot, Dank Memer)
Owners - Server owners (Straight From MG, RandomPerson3465, Sync)

The Regular, Contributor, and Server Booster roles do not give extra perks or permissions.

Selectable roles:

Server Updates - Pinged for announcements/changes about the Discord server
(LCEDIT) Announcements - Pinged for Livecountsedit announcements
(LCEDIT) Updates - Pinged for major Livecountsedit updates
(LCEDIT) Minor Updates - Pinged for smaller Livecountsedit updates (that add content and/or fix at least 5 bugs), in addition to major ones

(LCEDIT) Leaks - Pinged for leaks of future updates (Note: content in leaks are never 100% guaranteed to be released as an update)
(LCEDIT) Polls - Pinged for polls about Livecountsedit
Suggestion - A post that requests for a feature to be added, changed, or removed
Help - A post that asks for help on how to use the site, fix problems, or use certain features
Bug Report - A post that reports behavior that is clearly unintended (such as features not working correctly or at all, errors, layout problems)

Read the post guidelines of the appropriate channel before posting suggestions, help requests, or bug reports.

Do not ping owners for suggestions.

If you need to send a link (for legitimate reasons ONLY), send without the "https://" to prevent it from being blocked by automod.

If you need to send a save file for Top 50 or the Livecountsedit themed counter, you should send a public save, DO NOT send private saves.
